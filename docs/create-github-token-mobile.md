# Crée un GitHub Token (1 minute)

**Ouvre ce lien sur ton mobile:**

👉 **https://github.com/settings/tokens/new**

**Remplis:**

1. **Note:** `Mission Control Deploy`
2. **Expiration:** 30 days
3. **Scopes:** Coche uniquement ✅ **repo** (Full control of private repositories)

**Scroll down → Tap "Generate token"**

**IMPORTANT:** Copie le token (tu ne le verras qu'une fois!)

Il ressemble à: `ghp_xxxxxxxxxxxxxxxxxxxx`

---

## Ensuite, envoie-moi le token via Telegram:

```
GITHUB_TOKEN: ghp_xxxxxxxxxxxxxxxxxxxx
```

**Je vais pusher le code immédiatement!** (30 secondes)
