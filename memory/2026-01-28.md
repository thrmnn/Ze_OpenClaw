## Lessons Learned

### Communication
- Always confirm understanding immediately
- Don't disappear into long tasks without updates
- "Yo" means check in

### Privacy & Local Solutions
- User prefers local processing over cloud APIs (privacy)
- faster-whisper works well for voice transcription
- Worth the setup complexity for privacy guarantee

### WSL Environment
- Dependency hell with ML packages (transformers, torch)
- faster-whisper is more reliable than transformers pipeline
- Network isolation: Windows apps may not be accessible from WSL

### API Integration
- Direct REST API calls often better than MCP dependencies
- Test API connections before declaring success
- Document alternatives when primary solution fails
